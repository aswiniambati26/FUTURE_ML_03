# 📄 Resume Screening & Candidate Ranking System

## About the Project

Recruiters often receive hundreds of resumes for a single job opening, making the screening process time-consuming and challenging. Manually reviewing every application can lead to inconsistencies and delays in identifying the most suitable candidates.

In this project, I developed a Resume Screening and Candidate Ranking System that automatically analyzes resumes, extracts relevant skills, compares them with job requirements, and ranks candidates based on their suitability for a specific role.

The goal of this project is to demonstrate how Natural Language Processing (NLP) and Machine Learning techniques can be used to streamline recruitment processes and support data-driven hiring decisions.

---

## Problem Statement

Recruiters and hiring managers often face challenges such as:

- Large volumes of applications for a single job opening
- Time-consuming manual resume screening
- Difficulty identifying the most qualified candidates
- Inconsistent evaluation of applicants
- Missing suitable candidates due to human oversight

To address these challenges, this project automates the initial screening process by matching resumes with job descriptions and ranking candidates based on their relevance to the role.

---

## What I Did

- Processed resume and job description text data
- Performed text cleaning and preprocessing
- Extracted important skills and keywords
- Converted text into numerical representations
- Measured similarity between resumes and job requirements
- Ranked candidates based on matching scores
- Identified missing skills and skill gaps

---

## Project Workflow

### 1. Data Collection

Collected resume and job description data for analysis and comparison.

### 2. Text Preprocessing

Prepared the textual data by:

- Converting text to lowercase
- Removing punctuation and special characters
- Removing stopwords
- Cleaning and normalizing text

### 3. Skill Extraction

Extracted important skills and keywords from resumes and job descriptions.

Examples include:

- Python
- SQL
- Machine Learning
- Data Analysis
- Deep Learning
- Communication Skills

### 4. Feature Engineering

Converted textual information into machine-readable numerical representations using NLP techniques.

### 5. Resume Matching

Compared resume content against job requirements using similarity-based methods to determine how closely a candidate matches the desired role.

### 6. Candidate Ranking

Generated ranking scores based on:

- Skill relevance
- Keyword matching
- Resume-job similarity
- Overall role fit

### 7. Skill Gap Analysis

Identified:

- Matching skills
- Missing skills
- Areas where candidates can improve their profiles
